HTML Headings (<h1> to <h6>)

Headings are the tittles or Subtittles that you want to display on Webpage.

HTML provides six levels of headings (h1 to h6) arranged in order of priority, where h1 is the main title or has highest Priority and h6 is the least important, Lowest Priority.

HTML is a block-level element means it always starts on a new line in the webpage

You should use only one <h1> per page as it is the main title, while <h2> to <h6> can be used multiple times for sections and subsections.

Paragraph <p>

A paragraph always starts on a new line because <p> is a block-level element.

Multiple spaces or new lines inside <p> are ignored by the browser.

You can you <br> to break the line inside a pragraph.

If you want to add spaces or extra lines in your text, you can use <pre> instead of <p>, because <pre> keeps the formatting exactly as written.

Form Elements

1. <form> element is used to create an HTML form for user input:



2. The <label> Element

The <label> element is used to give a text description for an input element (like textbox, checkbox, radio button, etc.).
It improves usability and accessibility because clicking on the label also activates the input.



3. The <input> Element

The <input> element is used to create form fields where users can enter data or make a selection.
The behavior of <input> changes depending on its type attribute.


After writing this in a code you will get the below Output.

<label for="fname">First name:</label>
<input type="text" id="fname" name="fname">

for Attribute in <label>

The for attribute is used inside a <label> tag.
Its value must match the id of the input element it belongs to.

id Attribute in <input>

The id attribute inside an <input> uniquely identifies that input field.
It is mainly used to link the input with a <label> (via the for attribute).

How for and id Work together

The label has for="fname".
The input has id="fname".
Because the values match, they are linked.
Now when you click on the text "First Name:", the cursor automatically jumps into the textbox, and you can start typing right away.
if don't use for and id then clicking on “First Name” does nothing. You have to click directly inside the box.

name Attribute (inside <input>)

The name attribute is used by the backend to identify the data when the form is submitted.
It is the key in the "key-value" pair that gets sent to the server/backend.

placeholder Attribute

The placeholder shows a hint text inside the input box.
It disappears when the user starts typing.
It tells the user what kind of input is expected.
